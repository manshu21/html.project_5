Laundry Services Website

A clean and simple (Laundry Services landing page) built using HTML5 and CSS3.  
This project focuses on basic web layout concepts like navbar styling, inline-block, float positioning, and a hero section design without using Flexbox.

Features
1.Responsive-style layout using **inline-block & float**
2.Simple and clean **navigation bar**
3.Attractive **hero section** with call-to-action
4.Beginner-friendly structure
5.No frameworks used (Pure HTML & CSS)

Technologies Used
1.HTML5
2.CSS3
3.Google-safe fonts
4.Basic layout techniques (float, inline-block)

Page Sections

Navigation Bar
1.Logo on the left
2.Menu links in the center
3.User button on the right
4.Built using `float` and `inline-block`

Learning Purpose

This project is ideal for beginners to understand:
1.Difference between **inline, inline-block, and block**
2.How `float` works in layouts
3.Basic navbar design without Flexbox
4.Structuring a landing page
